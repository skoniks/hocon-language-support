# Change Log

All notable changes to the "hocon-language-support" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [Unreleased]

- Improved extension metadata in `package.json` (keywords, categories, project description)
- Updated project documentation in README

## [0.0.1]

- Registered HOCON language for `.conf` and `.hocon` files
- Added language configuration for brackets, auto-closing pairs, and surrounding pairs
- Added basic TextMate grammar for comments, strings, numbers, booleans, null, substitutions, operators, keys, and `include`
- Added minimal document formatter for HOCON via VS Code API
